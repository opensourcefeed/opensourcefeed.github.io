> **Note:** Raspbian is the former name commonly associated with Raspberry Pi OS. For the current official operating system, see [Raspberry Pi OS](/distribution/raspberry-pi-os/).

Raspbian was a Debian-based Linux distribution optimized for Raspberry Pi computers. It was created as an independent project to bring Debian to the Raspberry Pi platform with packages built for its ARM hardware.

For many years, Raspbian was the main operating system used by Raspberry Pi owners, students, hobbyists, and makers. It gave Raspberry Pi users access to the Debian software ecosystem while keeping the system lightweight enough for small single-board computers.

## Raspbian and Raspberry Pi OS

Raspbian is now mainly a historical name. The current official operating system for Raspberry Pi computers is called **Raspberry Pi OS**.

The name change helped Raspberry Pi separate the modern official operating system from the older Raspbian project, especially as Raspberry Pi OS expanded to newer hardware and 64-bit editions.

If you are setting up a Raspberry Pi today, you should usually download and use **[Raspberry Pi OS](/distribution/raspberry-pi-os/)** instead of looking for old Raspbian images.

## Key Features of Raspbian

Raspbian was known for its close connection to Debian and its focus on Raspberry Pi hardware. Its main strengths included:

- Debian-based package management
- Raspberry Pi hardware optimization
- Large software repository
- Lightweight desktop experience
- Strong community support
- Good fit for education, coding, electronics, and small home projects

## Why Raspbian Still Matters

Many older tutorials, books, forum posts, and project guides still use the name Raspbian. So the term remains useful when reading older Raspberry Pi documentation or maintaining older Raspberry Pi installations.

For modern use, however, Raspberry Pi OS is the better page to follow because it represents the actively maintained official operating system.
