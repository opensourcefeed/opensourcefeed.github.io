**Antergos NeXT** is a community-led revival of the original Antergos Linux distribution. Its goal is to preserve the polished, beginner-friendly experience that made Antergos popular while keeping pace with modern Arch Linux.

Built on Arch Linux, Antergos NeXT follows a rolling-release model, providing users with the latest kernels, drivers, and applications directly from upstream repositories without requiring major version upgrades.

One of its defining features is the return of the **Cnchi** graphical installer. The installer has been updated for compatibility with modern Python releases, restoring the simple installation workflow that distinguished the original Antergos project.

KDE Plasma serves as the default desktop environment and ships with the familiar Antergos visual identity, including its classic icon set and wallpapers. During installation, users can also choose from Cinnamon, XFCE, GNOME, Budgie, Deepin, LXQt, Openbox, and i3.

Although inspired by the original Antergos project, Antergos NeXT is developed and maintained by a separate community team. It should be viewed as a revival of the Antergos experience rather than a direct continuation of the original project.

For historical information about the original distribution, see the [**Antergos** distribution](/distribution/antergos) page.
