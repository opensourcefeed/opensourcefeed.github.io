**PureOS** is a Debian-based Linux distribution developed by Purism with a focus on privacy, security, and software freedom. The distribution is endorsed by the Free Software Foundation (FSF) because it ships entirely with free and open-source software.

PureOS is used on Purism's Librem hardware lineup, including Librem laptops, the Librem Mini, and the Librem 5 smartphone. It can also be installed on compatible desktop and laptop computers from other vendors.

The distribution uses the GNOME desktop environment and follows a clean and straightforward user experience. PureOS avoids proprietary software and tracking technologies by default, making it popular among users who prefer transparent and privacy-respecting systems.

PureOS focuses on:
- Privacy-focused computing
- Free software principles
- User control and transparency
- Security-conscious system design
- Open-source development
- Desktop and mobile Linux convergence

The distribution includes common Linux desktop applications for web browsing, office work, media playback, communication, and software development. It uses the APT package management system and benefits from Debian's large software ecosystem.

PureOS is closely connected to Purism's hardware and software ecosystem. On supported Librem devices, the operating system can integrate with Purism technologies such as:
- Physical hardware kill switches
- PureBoot security features
- Privacy-focused firmware initiatives
- Adaptive interfaces for desktop and mobile usage

The Librem 5 smartphone is one of the most visible PureOS-powered devices. Purism uses PureOS as the main operating system for both mobile and desktop products, with the goal of providing a more unified Linux experience across device types.

PureOS has gained attention in the Linux community for its strong stance on software freedom and privacy. The project is also part of the wider Debian and GNOME ecosystem, alongside related communities focused on ethical and open computing.

The distribution remains one of the best-known Linux options for users looking for a fully free operating system with a modern desktop experience and privacy-oriented defaults.
