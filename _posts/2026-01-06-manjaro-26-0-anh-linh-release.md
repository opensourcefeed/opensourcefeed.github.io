**The** Manjaro team has released **Manjaro 26.0**, code-named **Anh-Linh**. This version follows the April 2025 release of Zetar and brings updates across all major desktop editions.

Manjaro 26.0 ships with **Linux kernel 6.18** by default. For users who need long-term stability or support for older hardware, **kernel 6.12 LTS** and **6.6 LTS** remain available.

![Manjaro 26.0 release with code name Anh-Linh](/assets/images/post-images/manjaro/manjaro-26.0.jpg)

Below is a clear look at what’s new in this release.

---

## GNOME Edition: Based on GNOME 49

The GNOME edition now uses the **GNOME 49** series. This update focuses on stability, speed, and better display support.

### Key improvements

**Redesigned Calendar app**  
The Calendar app now has a flexible layout that adapts to different window sizes. Users can hide the sidebar when working on small screens.

**Faster Software app**  
The GNOME Software app now handles large Flatpak repositories more efficiently. Memory use is lower, and browsing feels faster, even on low-end systems.

**HDR wallpapers and better color support**  
GNOME 49 adds a new wallpaper collection made for HDR displays and the Display P3 color space. Mutter now supports full 16-bit color depth, which improves contrast and color accuracy. The image system, Glycin, also runs faster.

**Improved remote desktop features**  
GNOME’s built-in remote desktop tools now offer better support for users who access their system from another device.

---

## KDE Plasma Edition: Plasma 6.5, Frameworks 6.21, KDE Gear 25.12

The Plasma edition in Manjaro 26.0 ships with the **Plasma 6.5** desktop and the latest KDE software stack.

### What’s new in Plasma 6.5

**Rounded window corners**  
Breeze-themed windows now use rounded corners on all sides. Users who prefer sharp edges can turn this off.

**Automatic light and dark themes**  
Plasma can now switch themes based on the time of day. You can choose which themes are used and how wallpapers react to the change.

**Unified app permissions**  
The old Flatpak permissions page is now a full **Application Permissions** section. It lets you manage access to features like screenshots and remote control.

**Better remote desktop**  
Plasma’s RDP server now supports clipboard sharing. You no longer need to create special remote users. Existing system accounts work as expected.

**Printer alerts**  
The system now warns you when ink or toner runs low.

**Hibernate from the login screen**  
You can now hibernate the system directly from the login page.

---

## Xfce Edition: Now with Xfce 4.20

The Xfce edition moves to **Xfce 4.20**, bringing small but useful upgrades.

### Highlights

**File highlighting in Thunar**  
You can now set custom background and text colors for files. This helps when working in folders with many similar files. Thunar also adds recursive search.

**Panel improvements**  
Panel length is now set in pixels instead of percentages. There is also a new option to keep the panel above windows, even when apps are maximized.

**Updated Control Centre**  
All system settings are now grouped in one window. New options include:  
- Turning off header bars in dialogs  
- Showing or hiding the delete option in file menus  
- Choosing default behavior for multi-monitor setups  

---

## Kernel and Hardware Support

Manjaro 26.0 uses **Linux kernel 6.18**, which brings the latest drivers and hardware support.  
For users who prefer long-term stability, **6.12 LTS** and **6.6 LTS** kernels remain supported.

This gives users flexibility based on their system age and needs.
