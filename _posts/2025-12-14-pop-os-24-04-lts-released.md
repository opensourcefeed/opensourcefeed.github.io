System76 has released **Pop!_OS 24.04 LTS**, a long-term support version that introduces the new **COSMIC Desktop Environment**. This release marks a major technical shift for Pop!_OS and sets the foundation for future development.

![Pop!_OS 24.04 release](/assets/images/post-images/popos/24.04.webp)

## COSMIC Desktop: Epoch 1

COSMIC is a new desktop environment built fully by System76. The project began after the team reached the limits of their previous desktop stack. COSMIC is designed to be modular, flexible, and easy to extend.

It serves as the default desktop for Pop!_OS, while remaining adaptable for other Linux projects.

## Pop!_OS 24.04 LTS Highlights

- New COSMIC desktop experience
- ARM support
  - Official support for System76 Thelio Astra
  - Community support for other ARM devices using Tow-Boot
- Improved hybrid graphics handling
  - No manual GPU switching
  - Apps select the correct GPU automatically
  - Users can choose a GPU per app using right-click options
- Simple installation with full disk encryption
- Refresh install feature
  - Reinstall while keeping files, settings, and Flatpak apps
  - Available from the installer or by holding Space during boot
- Broad hardware compatibility

## COSMIC Desktop Features

### Window Tiling and Workspaces

COSMIC includes built-in tiling that works with both mouse and keyboard.

- Toggle tiling from the panel
- Configure tiling per workspace or per display
- Drag windows with clear visual placement guides

Workspaces are flexible and persistent.

- Horizontal or vertical layouts
- Per-display or shared workspaces
- Reorder or move workspaces between displays
- Pin workspaces so they remain after reboot
- Optional numbered workspace applet

### Multi-Display Support

COSMIC handles mixed-resolution displays smoothly.

- Automatic scaling for HiDPI and standard screens
- Manual scaling adjustments in Settings
- Display settings persist across reconnects
- Windows move automatically when a display is removed

### Customization Options

Users can customize layout and appearance without extra tools.

- Theme colors set from Settings
- Panel and Dock or single Panel layout
- Place panels on any screen edge
- Add and arrange applets from Settings

### Navigation and Keyboard Shortcuts

- Press **Super** to open the launcher
- Type to open or switch applications
- Built-in tools include search, calculator, and command runner
- Use Super + Arrow keys to move or focus windows

### Window Stacks and Snapping

- Stack windows into tab groups
- Snap floating windows to screen edges
- Set sticky windows that follow across workspaces

### COSMIC Applications

Pop!_OS 24.04 LTS includes core COSMIC apps:

- COSMIC Files
- COSMIC Store
- COSMIC Terminal
- COSMIC Text Editor
- COSMIC Media Player
- COSMIC Screenshot

All COSMIC components are written in **Rust**, focusing on safety and performance.

## Long-Term Support

Pop!_OS 24.04 is a long-term support release. It forms the base for future COSMIC updates and ongoing improvements. Development of COSMIC is funded through System76 hardware sales.

Systems running Pop!_OS 22.04 LTS will receive an upgrade notification starting **January 2026**. Users who want to upgrade earlier can do so manually after backing up their data.
