**FreeBSD** 15.0 is now available. This release brings updated drivers, new hardware support, and changes that make the system more stable. It continues the long line of FreeBSD releases that aim for clarity and reliability.

![FreeBSD 15 RELEASE](/assets/images/post-images/freebsd/freebsd-15.jpg)

## What's new in FreeBSD 15.0

FreeBSD 15.0 includes changes across the system. These updates improve hardware support, development tools, storage, networking, and security.

### System changes
- The release removes the old 32-bit Tier 1 support on x86. It now focuses on 64-bit platforms.
- ARM and RISC-V platforms see improvements and better support.
- The base system includes refreshed libraries and tools.

### Hardware and driver updates
- More graphics cards are supported.
- Storage and network drivers get updates for modern hardware.

### Security improvements
- The release includes fixes from recent security advisories.
- Cryptography tools receive updates to meet current standards.

### Development tools
- LLVM and Clang receive updates.
- The system ships with refreshed build tools and compilers.

These updates help developers and system administrators who depend on FreeBSD for reliable performance.
