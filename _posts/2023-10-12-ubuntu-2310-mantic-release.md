**On** 12th October 2023, the Canonical has announced the release of Ubuntu 23.10 <em>Mantic Minotaur</em>. Ubuntu 23.10 is a regular release from the world's famous GNU/Linux distribution, and it will have a support period of 9 months. The release also includes other official community flavors of Ubuntu, including Xubuntu, Lubuntu, Ubuntu MATE, Ubuntu Cinnamon, Ubuntu Budgie, Kubuntu, and Ubuntu Studio.

![Ubuntu Mantic featured image](/assets/images/post-images/ubuntu-mantic/featured.jpg)

## What is new in Ubuntu 23.10 Mantic Minotaur?

Ubuntu - or the official Ubuntu release with the GNOME shell includes several noteworthy changes.

> - Ubuntu 23.10 features GNOME 45 - the latest stable release from the popular desktop enviornment. GNOME 45 further enhances user experience and productivity.
- Ubuntu Mantic introduces the new App Center, which replaces Ubuntu Software. It offers a faster and smoother way for installing snap and debian packages.
- Firmware updater is now independent from the App Center. This reduces the background load when checking for new firmware updates.
- Ubuntu Desktop 23.10 introduces an experimental feature to enhance existing Full Disk Encryption (FDE) functionality by supporting Trusted - Platform Modules (TPMs) for key storage. 
- Ubuntu Mantic introduces new administration tools as part of Ubuntu Pro. It will be free for every user on up to 5 machines.
- Ubuntu Mantic further improves the Subiquity installer - which was introduced in Ubuntu 23.04 release.

![Ubuntu 23.10 Mantic Minotaur](/assets/images/post-images/ubuntu-mantic/ubuntu.webp)

For [further information on Ubuntu 23.10](https://canonical.com/blog/ubuntu-desktop-23-10-mantic-minotaur-deep-dive) read the official release announcement.

## What's new in Ubuntu Budgie 23.10 Mantic Minotaur?

Ubuntu Budgie 23.10 - the Ubuntu flavor with the Budgie Desktop - brings the GNOME 45 stack integration.
> - Budgie Desktop v10.8 enhances the menu, introduces status notifier support on system tray, and other improvements.
- Updates the Budgie Control Center to incorporate the enhancements and bug fixes implemented in past six months.
- Updates all theme packages.
- Budgie Welcome is updated to make things better.

![Ubuntu Budgie 23.10 Mantic Minotaur](/assets/images/post-images/ubuntu-mantic/ubuntu-budgie.jpg)

For [further reading on Ubuntu Budgie 23.10](https://ubuntubudgie.org/2023/09/ubuntu-budgie-23-10-release-notes/) - see the official release announcment.

## What's new on Xubuntu 23.10?

Xubuntu 23.10 inherits all core updates from Ubuntu 23.10 and it uses updated packages from GNOME 45 and Xfce 4.18.

![Xubuntu 23.10 Mantic Minotaur](/assets/images/post-images/ubuntu-mantic/xubuntu-23-10.png)

To understand [the new features on Xubuntu 23.10 in details](https://blog.bluesabre.org/2023/10/10/whats-new-in-xubuntu-23-10/), read the blog post by Sean Davis.

## What's new on Lubuntu 23.10?
Lubuntu - the Ubuntu flavor with LXQt desktop brings following updates with Mantic release.

> - LXQt 1.3.0
- Qt 5.15.10
- LibreOffice 7.6
- VLC 3.0.18
- Featherpad 1.3.5
- Discover Software Center 5.27.8

For [further reading on Lubuntu 23.10](https://lubuntu.me/mantic-released/) - read the official release announcement.

## What is new on Ubuntu MATE 23.10?

Ubuntu MATE 23.10 - the Ubuntu release with MATE desktop - offers a traditional user experience with MATE Desktop 1.26.2. It also includes several AI generated wallpapers with the spirit of Ubuntu Mantic and MATE Desktop. It also includes celluloid 0.25, Evolution 3.50, LibreOffice 7.6.1

![Ubuntu MATE 23.10 Mantic Minotaur](/assets/images/post-images/ubuntu-mantic/ubuntu-mate.webp)

For [further information on Ubuntu MATE 23.10](https://ubuntu-mate.org/blog/ubuntu-mate-mantic-minotaur-release-notes/), see the official release notes published on projects blog.

## What is new on Ubuntu Cinnamon 23.10?

Ubuntu Cinnamon 23.10 is the second release from the project as the official community flavor. The noteworthy updates in Ubuntu Cinnamon 23.10 are as follows.

> - Cinnamon 5.8.4
- Nemo 5.8.4
- Plymouth theme updates
