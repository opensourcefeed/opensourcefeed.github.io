**The** Sparky Linux team has announced the release of Sparky Linux 8.2. This is the second quarterly update of the Sparky 8 “Seven Sisters” stable series.

![Sparky Linux 8.2 release with Debian Trixie base](/assets/images/post-images/sparky/sparky-8.2.jpg)

The release is based on and fully compatible with Debian Project Debian 13 “Trixie”.

This update focuses on package refresh, desktop updates, kernel improvements, and a small bug fix.

## Main Changes in Sparky 8.2

> * Updated packages from Debian and Sparky stable repositories (as of 14 Feb 2026)
* Default Linux kernel **6.12.69-LTS**
  * Extra kernels available in Sparky repositories
* The Document Foundation LibreOffice **25.2.3**
* KDE Plasma **6.3.6**
* LXQt Project LXQt **2.1.0**
* MATE Desktop Project MATE **1.26.0**
* Xfce Project Xfce **4.20**
* Openbox Project Openbox **3.6.1**
* Mozilla Foundation Firefox **140.7.0esr**
* Thunderbird **140.7.1esr**
* Fixed an Openbox issue that blocked terminal launch from the package manager

## Available Editions

### PC (amd64 BIOS/UEFI + Secure Boot)

* Xfce
* LXQt
* MATE
* KDE Plasma
* MinimalGUI (Openbox)
* MinimalCLI (text mode)

### ARM64

* Openbox desktop
* CLI edition

## Default Login Details

* **PC live session:** `live : live`
* **ARM edition:** `pi : sparky`

For [further information and download links of Sparky Linux 8.2](https://sparkylinux.org/sparky-8-2/), see the official release announcement.
