**Mauna Linux 25.1** is officially released as a **minor update** to the Mauna 25 series. Existing Mauna 25 users will receive notifications for updates, which can be installed via the Mauna Update application.

![Mauna Linux 25.1 featured image](/assets/images/post-images/mauna/mauna-25.1.jpg)

This release brings updates to desktop environments, applications, firmware, and system tools.

## What's New in Mauna Linux 25.1

- **OnlyOffice 9.2.0** is included as the default office suite.
- Firmware updated to version **20251111**.
- Xfce AHE now uses **Linux Kernel 6.17.8**.
- Debian base updated to **13.2 Trixie**.
- Firefox 146.0 and Librewolf 146.0 included.
- Cinnamon 6.4.10 desktop in Mauna Christian Edition 25.1.
- About Mauna 1.1.1 and system applications updated.
- GNOME Desktop temporarily unavailable due to installation issue.


## Firmware Updates

The firmware collection is updated to the latest version as of **November 11, 2025 (20251111)**.

## Xfce AHE (Advanced Hardware Enablement)

The Xfce AHE edition now uses **Linux Kernel 6.17.8** as the default. This kernel may receive future updates.

## Debian Base

Mauna 25.1 is based on **Debian 13.2 Trixie**, providing a stable and secure system foundation.

## Firefox and Browser Updates

The default browser remains **Firefox 146.0**, and the system also includes **Librewolf 146.0**.


## Mauna Christian Edition 25.1

The Christian Edition now features the **Cinnamon 6.4.10 desktop** and all standard system updates.

## GNOME Desktop Status

The GNOME Desktop is temporarily unavailable due to an installation issue. No estimated time for restoration has been announced.
